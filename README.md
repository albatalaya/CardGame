# CardGame

BlackJack Game for the JavaScript project.

## Built with
- Vite
- Vanilla JavaScript

## Getting started
* Clone this repository: `git clone https://github.com/albatalaya/CardGame.git.
* `cd BlackJack`
* `npm install` 
* `npm run build` 
* `npm run dev` 
* Go to http://localhost:3000/ to see the result
